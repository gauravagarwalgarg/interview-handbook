## How to Use

1. **Before the interview** Review the level calibration for the target role
2. **During the interview** Use the round-specific rubric to score in real-time
3. **After the interview** Fill out the feedback template within 24 hours
4. **In debrief** Reference calibration matrix to justify your hire/no-hire decision

## Scoring Scale (Universal)

| Score | Label | Definition |
|-------|-------|------------|
| 1 | Strong No-Hire | Significant gaps; would not succeed at any level |
| 2 | No-Hire | Below bar for target level; gaps in critical areas |
| 3 | Lean Hire | Meets bar with minor concerns; would grow into role |
| 4 | Strong Hire | Clearly exceeds bar; immediate impact expected |
